# Graph Proof Demo

Interactive HTML tool for presenting and exploring proper edge-colorings on the diamond-chain graph used in the proof.

## Features

- Click an edge to see the colors currently allowed there.
- Proper edge-colorings are enforced automatically.
- Click a node to highlight all incident edges for presentation.
- Add or remove diamonds in the chain with the compact sidebar controls.
- Reset all edge colors at any time.

## Usage

Open `graph_proof.html` in a browser, or serve the folder locally and visit the page there.
